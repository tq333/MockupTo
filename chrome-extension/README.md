# Mockup Sync · Chrome Extension

Browser side of Mockup Sync. Captures any Mockup Kit demo page into IR JSON
that the Figma plugin can re-render.

## Build

```bash
cd mockup-sync/chrome-extension
npm run build          # src/capture.src.js + ../mockup-kit.mapping.json → lib/capture.bundle.js
npm run check          # syntax-check the source (Node parser, no execution)
```

The build is self-contained — it reads `src/capture.src.js`, injects the
mapping JSON in place of the `/* @inject:mapping */ null` marker, and writes
`lib/capture.bundle.js`. There is no separate bookmarklet build step.

## Install (developer mode)

1. Open `chrome://extensions`
2. Enable **Developer mode** (top right)
3. Click **Load unpacked** → select this folder (`mockup-sync/chrome-extension`)

## Use

* Click the toolbar icon to open the popup, then **Capture this page**.
* Or press **⌘⇧M** (Mac) / **Ctrl+Shift+M** (others) — captures without opening
  the popup. Output behavior follows the popup options (auto-copy, download,
  toast).

## Files

```
manifest.json              MV3 manifest, action popup, command shortcut
background/
  service-worker.js        Handles the ⌘⇧M command (capture without popup)
popup/
  popup.html / .css / .js  UI: capture button, options, history list
src/
  capture.src.js           ← editable source (DO edit this, NOT lib/)
lib/
  capture.bundle.js        ← generated by `npm run build` (DO NOT edit)
icons/
  icon-{16,48,128}.png     ← generated placeholder; replace with real art
scripts/
  build.mjs                Bundles src/ + mapping → lib/
```
